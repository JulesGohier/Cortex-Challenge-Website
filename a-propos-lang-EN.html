Welcome to the presentation page of our realization project.

This project aimed to put into practice all the programming concepts seen during our 1st year of computer science studies. It is divided into two parts, an algorithmic part aimed at solving a given problem in Python and a web development part in which the project itself is explained. Through this project, we were able to apply the notions studied in class and deepen them. In addition, our ability to search for information in technical documentation (Stack Overflow, MDN Web docs, W3Schools), our teamwork, and our project management skills (Git, GitHub) have improved. Finally this project allowed us, at the same time, to evolve on all levels and to have an overview of what it is to work in a group to carry out a project.
